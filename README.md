# Coffee Shop App 

A basic Flutter coffee shop demo application focused on UI design and state management.

## Features
- Product listing screen
- Add to cart functionality
- Cart item count update
- Total price calculation
- Basic state management using setState / GetX

## Tech Stack
- Flutter
- Dart
- Firebase (basic understanding)
- GetX (state management)

## Purpose
This project was created as a demo application to practice Flutter UI development and state management concepts.
